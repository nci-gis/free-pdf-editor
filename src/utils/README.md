# Utilities Documentation

This folder contains utility modules for the PDF Editor static web-app. Below is a summary of each module and its exported functions:

## asyncReader.js
- `readAsArrayBuffer(file)`: Reads a file as an ArrayBuffer.
- `readAsImage(src)`: Loads an image from a source URL.
- `readAsDataURL(file)`: Reads a file as a DataURL.
- `readAsPDF(file)`: Reads a PDF file and returns a PDF object.

## helper.js
- `ggID()`: Generates unique IDs.
- `timeout(ms)`: Returns a promise that resolves after a given time.

## pannable.js
- `pannable(node)`: Svelte action for drag/pan events.

## PDF.js
- `save(pdfFile, objects, name)`: Saves the edited PDF.

## prepareAssets.js
- `getAsset(name)`: Gets an asset by name.
- `prepareAsset({ name, src })`: Prepares a single asset.
- `prepareAssets()`: Prepares all assets.
- `fetchFont(name)`: Loads a font by name.

## tapout.js
- `tapout(node)`: Svelte action for tap-out events.
