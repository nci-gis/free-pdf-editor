# Free PDF Editor

## Static Deployment Instructions

1. Open `public/index.html` directly in Chrome or Edge.
2. All features work client-side; no backend required.
3. For offline use, ensure all assets in `public/` and `src/` are present.

## Project Structure

- `src/components/`: Svelte UI components (Toolbar, PDFPage, Drawing, etc.)
- `src/utils/`: Utility modules (see `src/utils/README.md`)
- `src/assets/`: Static assets (images, icons, etc.)
- `public/`: Entry HTML and build output
- `docs/`: Project documentation and analysis
